# Изображения для лендинга

Поместите следующие изображения в папку `public/`:

## Hero секция (главный баннер)
- **og-image.jpg** - главное изображение для Hero секции (рекомендуемый размер: 1200x709px)

## Портфолио офисов
- **teniz-towers.jpg** - изображение для Teniz Towers (рекомендуемый размер: 800x600px)
- **orion.jpg** - изображение для Orion (рекомендуемый размер: 800x600px)
- **nurlytau.jpg** - изображение для Nurlytau (рекомендуемый размер: 800x600px)
- **venus.jpg** - изображение для Venus (рекомендуемый размер: 800x600px)

## Формат файлов
- Формат: JPG или PNG
- Оптимизируйте изображения для веба (можно использовать инструменты типа TinyPNG)

После добавления изображений проект автоматически будет их использовать.
